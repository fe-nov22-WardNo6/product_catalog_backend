Documentation of product_catalog API

BASE_URL: https://product-catalog-backend-vhc5.onrender.com

1. Get: /phones

Return list of phones.

  • - Without search params: get collection of first 16 phones.

    Example: BASE_URL/phones

  • - With pagination: required url search params: perPage and currentPage.

    Example: BASE_URL/phones?perPage=5¤tPage=2

  • - With sort: required url search params: sortBy. Availeble name of collection ['cheapest', 'expensive', 'newest']

    Example: BASE_URL/phones?sortBy=cheapest

2. Get: /phones/:phoneId

Return extend information of phone.

  • - Required params "phoneUrl"

    Example: BASE_URL/phones/phoneId

3. Get: /phones/collection/:collectionName

Return collection of 8 items in available categories.

  • - Required params "collectionName", Availeble name of collection ['recommended', 'hotPrice', 'newest']

    Example: BASE_URL/phones/collection/recommended

4. Get: /downloads

Return image of item.

  • - Required search params "url"

    Example: BASE_URL/download?url=img/phones/apple-iphone-7/black/00.jpg

5.Get: /categories

Return all categories.

  • Example: BASE_URL/categories

6.Get: /count/:category

Return count of all items in category.

  • Example: BASE_URL/count/phones
